## Компилятор языка O на Java (17)
Язык O - подмножество языка Оберон. Доступный тип данных - integer, массивы, включая многомерные. 
Конструкции: for, while, repeat-until, if-elsif-else. 
Операции: сложения (+), разность (-), деление (MOD, DIV), умножение (*), операции сравнения. 
Стандартные процедуры и процедуры-функции языка: 
 - ABS - абсолютное значение
 - DEC - уменьшение числа на единицу
 - INC - увеличение числа на единицу
 - HALT - завершение программы с заданным кодом завершения
 - MIN - минимальное из двух чисел
 - MAX - максимальное из двух чисел
 - ODD - нечетное ли число  

Стандартные модули: 
 - IN - модуль для ввода данных из консоли
 - OUT - вывод в консоль с форматированием  

Все программы, написанные на языке O, будут рапускаться и на Оберон. Резрешен импорт только стандартных модулей, 
из-за чего программа состоит только из 1 файла.  

Подробное описание языка описано в книге Свердлова Сергея Залмановича "Языки программирования и методы трансляции".
